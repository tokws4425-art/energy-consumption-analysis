# Energy Consumption Analysis

## Description
This project analyzes electricity consumption data using Python.

## Features
- Identifies peak consumption periods
- Calculates average, minimum, and maximum consumption
- Visualizes energy usage trends

## Technologies
- Python
- pandas
- matplotlib

## Purpose
The goal of this project is to better understand energy consumption patterns.

## How to run

1. Install Python
2. Install required libraries:
   pip install pandas matplotlib
3. Run the script:
   python analysis.py
